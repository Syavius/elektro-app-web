# Прайс Електромонтаж — PRO Business UA

Застосунок для управління кошторисами та рахунками для електромонтажних робіт.

## Функціональність

- 📋 **Управління клієнтами** - база клієнтів з контактними даними
- 🛠️ **База матеріалів** - каталог матеріалів з цінами
- 💰 **Кошторис** - розрахунок вартості робіт та матеріалів
- 🧾 **Рахунки** - автоматичний розрахунок податків
- 💾 **LocalStorage** - збереження всіх даних в браузері
- 🖨️ **Друк/PDF** - експорт рахунків

## Податкові системи

- ФОП 2 група (єдиний податок)
- ФОП 3 група 5% без ПДВ
- ФОП 3 група 3% з ПДВ
- ТОВ на загальній системі
- Платник ПДВ (20%)
